# D&D 5e 角色创建与管理工具 🎲

一个功能强大且极具设计感的 **龙与地下城第五版 (D&D 5e)** 角色创建与管理系统。该工具支持从零建立角色、自选种族/职业、自定义属性、细节设定、法术书管理以及导出角色卡，并无缝集成了 5etools 中文镜像 (dickytwister) 数据库链接，是 TRPG 玩家与地下城主 (DM) 的绝佳助手。

---

## 🌟 核心功能

*   **🧙‍♂️ 起源与种族选择**：支持 D&D 5e 核心种族（如矮人、精灵、人类、龙裔、侏儒等）及其亚种选择，自动带入种族特性及技能熟练。
*   **⚔️ 职业与子职业拓展**：支持核心施法者（牧师、法师、吟游诗人等）及物理系职业，根据等级自动计算生命值上限、熟练加成与职业专长。
*   **📜 背景与技能熟练**：自定义角色成长背景，支持自由选择各种背景所赋予的技能、工具熟练和额外语言包。
*   **📊 属性值精细配置**：支持 **购点法 (Point Buy)** 和 **标准阵列 (Standard Array)**。动态扣减购点配额，并支持属性增加值 (ASI) 特性选择。
*   **✨ 法术管理与自选**：专门针对施法职业设计的法术配置面板，包含了全量 5e 核心法术数据库。支持自选洗练、戏法及高阶准备法术，同时支持玩家自定义填写非标准法术。
*   **💾 角色卡本地保存**：全本地浏览器存储 (`localStorage`)，即装即用。支持角色一键本地存储、多存档切换以及无感删除。
*   **📈 一键升级/降级**：可随时从编辑模式或角色卡界面调整角色等级，系统将自动重新分配技能槽位、重算 HP 及高环位法术槽。
*   **🔗 5etools 数据库直连**：在每个技能、特性和法术旁展示典雅的 **Wiki 小书标**，一键直达 [DickyTwister - 5etools] 对应的规则词条页面，方便实时查阅。

## 🛠️ 技术栈

*   **框架**：[React 19](https://react.dev/) + [Vite](https://vite.dev/) — 极致的构建速度与组件化开发体验。
*   **类型安全**：[TypeScript](https://www.typescriptlang.org/) — 全流程强类型约束，提升代码健壮性。
*   **样式**：[Tailwind CSS v4](https://tailwindcss.com/) — 语义化的原子类布局，深色羊皮纸风格與高级暗金相得益彰。
*   **动画**：[Motion (formerly Framer Motion)](https://motion.dev/) — 顺滑的步骤切换、弹出过渡，赋予工具生命的律动。
*   **图标**：[Lucide React](https://lucide.dev/) — 精致简约、高对比度的 SVG 图标。

## 🚀 快速上手

### 1. 安装依赖

在项目根目录下，执行以下命令安装运行所需依赖：

```bash
npm install
```

### 2. 启动本地开发服务

启动本地服务（默认端口为 `3000`）：

```bash
npm run dev
```

打开浏览器访问 `http://localhost:3000` 即可开始创建属于你的传奇角色。

### 3. 构建生产版本

静态资源将打包并输出至 `dist/` 目录：

```bash
npm run build
```

```text

│   ├── components/       # 交互组件与页面布局
│   │   ├── steps/        # 角色创建第一至第七步组件

│   │   ├── CharacterSheet.tsx   # 典雅的最终角色卡面板
│   │   └── LandingPage.tsx      # 一目了然的存档主页

│   ├── data/             # 详尽的 5e 核心数据（种族、职业、法术等）
│   ├── types/            # 完整的 TypeScript 类型与接口定义
│   └── main.tsx          # 渲染入口
├── package.json          # 依赖管理与运行脚本配置
└── tailwind.config.ts    # 页面整体风格与字族定义
```
