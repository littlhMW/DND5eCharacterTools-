# Littlh的DND5E角色创建工具 🎲

一个功能强大且极具设计感的 **龙与地下城第五版 (D&D 5e)** 角色创建与管理系统。该工具支持从零建立角色、自选种族/职业、自定义属性、细节设定、法术书管理以及导出角色卡。

无缝集成了 5etools 中文镜像 (dickytwister) 数据库链接，并已**完美支持多种扩展资源**（如《珊娜萨的万事指南》 XGE、《塔莎的万物坩埚》 TCE 等），是 TRPG 玩家与地下城主 (DM) 的绝佳车卡助手。

---

## 🌟 核心功能

*   **🧙‍♂️ 起源与种族选择**：支持 D&D 5e 核心种族及其亚种选择，自动带入种族特性及技能熟练。完美支持多种扩展资源的来源溯源。
*   **⚔️ 职业与子职业拓展**：支持各类施法者及物理系职业，根据等级自动计算生命值上限、熟练加成与职业专长。
*   **📜 背景与技能熟练**：自定义角色成长背景，支持自由选择各种背景所赋予的技能、工具熟练和额外语言包。已修正扩展专长的正确引流。
*   **📊 属性值精细配置**：支持 **购点法 (Point Buy)** 和 **标准阵列 (Standard Array)**。动态扣减购点配额，并支持属性增加值 (ASI) 特性选择与半专长计算。
*   **✨ 法术管理与自选**：包含海量扩展法术数据库。针对施法职业设计的法术配置面板，支持自选戏法及高阶准备法术，同时支持玩家自定义填写非标准法术。
*   **💾 角色卡本地保存**：全本地浏览器存储 (`localStorage`)，即装即用。支持角色一键本地存储、多存档切换以及无感删除。
*   **📈 一键升级/降级**：随时从编辑模式或角色卡调整等级，系统自动重新分配技能槽位、重算 HP 及高环位法术槽，增删相关特性。
*   **🔗 5etools 数据库扩展直连**：每个能力、种族、子职业及法术旁边提供 **Wiki 小书标**。**支持跨扩展包跳转**，不论是 PHB 还是 XGE、TCE，链接自动带入对应资源后缀（如 `_phb`, `_xge`, `_tce`），保证词条精确匹配。

## 🛠️ 技术栈

*   **框架**：[React 19](https://react.dev/) + [Vite](https://vite.dev/) 
*   **类型安全**：[TypeScript](https://www.typescriptlang.org/)
*   **样式**：[Tailwind CSS v4](https://tailwindcss.com/) — 深度融合于羊皮纸复古风格与高级排版。
*   **动画**：[Motion](https://motion.dev/) — 顺滑的步骤切换、弹出过渡。
*   **图标**：[Lucide React](https://lucide.dev/)

## 🚀 快速上手

### 1. 安装依赖

```bash
npm install
```

### 2. 启动本地开发服务

启动本地服务（默认端口为 `3000`）：

```bash
npm run dev
```

打开浏览器访问 `http://localhost:3000` 即可开始体验。

### 3. 构建生产版本

```bash
npm run build
```

## 📂 扩展与数据支持

本项目中的 `src/data` 目录下包含多种模块的规则数据，并按照书籍来源扩展划分（如 `phb/`, `xge/`, `tce/`, `vrgr/`, `vgm/`, `eepc/`等）：
- `races.ts`, `classes.ts`, `backgrounds.ts`, `feats.ts`, `spells.ts`
- **新增：** 塔莎的万物坩埚 (TCE)、珊娜萨的万事指南 (XGE) 的扩展法术与子职业（包含奇械师）数据。
- **新增：** 范·里希腾的鸦阁指南 (VRGR) 全新种族“半血裔”、“巫咒之子”与“重生者”，全面补全了不死本质、先祖遗赠、蛛行、吸血啃咬、怪诞徽记等深层种族特性背景，并依照 PHB 格式规范补充了其“本相”亚种结构。同时包含其专属子职业数据：异境吟游诗人（灵魂学院）、精魂德鲁伊（星辰德鲁伊形态等选择）与死灵主宰邪术师。
- **新增：** 邪恶元素玩家指南 (EEPC) 种族“元素裔”及 4 个亚种的深度描述；全面汉化并补充了《波洛的怪物指南》(VGM) 中“阿斯莫”、“斑猫人”、“纯血蛇人”、“费尔伯格”、“歌利亚”的深度起源故事与特性背景。
- **新增：** 全功能支持 VGM 的经典怪物可玩种族：“大地精”、“地精”、“狗头人”与“兽人”，集成完整的属性调整值、语言、体型，以及其标志性战斗与辅助特性（如群体战术、弱者发怒、挽回颜面、摇尾乞怜、侵略性、原初直觉等），并全面配予符合 150~200 字深度的沉浸式纯中文背景描述。
- 所有数据实体均具备 `source` 字段，以确保工具可以在不同扩展集之间清晰溯源并准确跳转到相应的中文维基节点。
- 所有表格展示已被优化为易读的 Markdown 纯文本列表，进一步适配本地数据展示。
